คณะในมหาวิทยาลัยศิลปากร วิทยาเขตสนามจันทร์

🎓 คณะศึกษาศาสตร์

คำอธิบาย: ผลิตครูและบุคลากรทางการศึกษา

สาขา: ภาษาไทย / ภาษาอังกฤษ / ปฐมวัย / ประถมศึกษา / เทคโนโลยีการศึกษา

🔬 คณะวิทยาศาสตร์

คำอธิบาย: ศึกษาวิทยาศาสตร์พื้นฐานและประยุกต์

สาขา: คณิตศาสตร์ / ฟิสิกส์ / เคมี / ชีววิทยา / เทคโนโลยีชีวภาพ / วิทยาการคอมพิวเตอร์

⚙️ คณะวิศวกรรมศาสตร์และเทคโนโลยีอุตสาหกรรม

คำอธิบาย: ออกแบบ พัฒนาเครื่องจักร และเทคโนโลยีอุตสาหกรรม

สาขา: วิศวกรรมเครื่องกล / ไฟฟ้า / เคมี / อุตสาหการ / วัสดุ / เมคคาทรอนิกส์

🌾 คณะสัตวศาสตร์และเทคโนโลยีการเกษตร

คำอธิบาย: การเกษตรสมัยใหม่ การผลิตอาหาร และการเลี้ยงสัตว์

สาขา: สัตวศาสตร์ / เทคโนโลยีการผลิตพืช / เทคโนโลยีอาหาร

💼 คณะวิทยาการจัดการ

คำอธิบาย: บริหารธุรกิจ การตลาด และการบริการ

สาขา: การตลาด / การจัดการ / การบัญชี / การท่องเที่ยว / โลจิสติกส์

💻 คณะเทคโนโลยีสารสนเทศและการสื่อสาร

คำอธิบาย: เทคโนโลยีดิจิทัล การเขียนโปรแกรม และสื่อออนไลน์

สาขา: เทคโนโลยีสารสนเทศ / วิทยาการข้อมูล / มัลติมีเดีย / นิเทศศาสตร์ดิจิทัล
